# Korean Delivery

## Current Rule

이제 기본 메인 보고서 자체를 한국어로 작성한다.
별도의 한국어 파생본을 기본 전제로 두지 않는다.

## What This Means

- 처음부터 한국어 메인 보고서 1본 작성
- 상세 CSV나 부속 자료는 필요하면 그대로 연결
- 기술 ID, 파일 경로, dataset/component/transaction 식별자는 backtick 유지

## If English Is Needed Later

영문이 필요하면 그때 한국어 메인 보고서와 근거 자료를 바탕으로 파생 작성한다.
하지만 기본 운영은 한국어다.
