# Samples

`samples/`는 본체가 아니라 검증용 입력셋 보관 위치다.

현재 포함된 `ScoreRanking_Proj-master`는 다음을 확인하기 위한 PoC 샘플이다.

- MiPlatform 화면 XML 파싱 가능 여부
- dataset / transaction 추출 가능 여부
- Spring Controller / Service / SQL map 연결 추적 가능 여부
- 표준 산출물 템플릿에 샘플 결과를 흘려보낼 수 있는지 여부

주의:

- 이 샘플의 구조를 본체 아키텍처 정답으로 삼지 않는다.
- 이 샘플 도메인을 MES 기준 도메인으로 삼지 않는다.
- 샘플은 분석 규칙과 자동화 흐름 검증용이다.
